

## Muscle Groups
When adding or modifying UI components that require a list of muscle groups, ALWAYS use the following complete and ordered list:
- chest
- back
- shoulders
- quads
- hamstrings
- glutes
- adductors
- legs
- biceps
- triceps
- abs
- calves
- forearms
- neck
- hip flexors
- full body
- other
Do not use older, shorter lists.
